IntervalArithmetic¶

Overview¶

IntervalArithmetic is a free and open source (*) interval arithmetic package for Python.

(*) IntervalArithmetic is licensed under GPLv3 therms.

IntervalArithmetic requires Python 3 and works on Linux, Windows and OS X.
