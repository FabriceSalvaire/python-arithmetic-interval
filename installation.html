3. Installation¶

Note to Packagers: Please don’t create IntervalArithmetic package (PiPY do the job)

3.1. On Windows¶

Firstly, you have to install the Anaconda Distribution so as to get a full featured Python 3 environment.

Then open the Anaconda Navigator and launch a console for your root environment.

You can now run pip to install IntervalArithmetic in your root environment using this command:

pip install IntervalArithmetic

3.2. On Linux¶

Firstly, you have to install Python 3 from your distribution.

Then you can install IntervalArithmetic using pip or from source. See supra.

3.3. On OSX¶

There are several ways to get Python on OSX:

  • use the built in Python
  • install Miniconda
  • install the Anaconda Distribution.
  • install from Brew brew install python3 (reported to work)

You can install IntervalArithmetic using pip or from source. See supra.

3.4. Installation from PyPi Repository¶

IntervalArithmetic is available on the Python Packages PyPI repository at https://pypi.python.org/pypi/IntervalArithmetic

Run this command in the console to install the latest release:

pip install IntervalArithmetic

3.5. Install a more recent version from Github¶

If you want to install a version which is not yet released on Pypi, you can use one of theses commands to install the stable or devel branch:

pip install git+https://github.com/FabriceSalvaire/IntervalArithmetic

pip install git+https://github.com/FabriceSalvaire/IntervalArithmetic@devel

3.6. Installation from Source¶

The IntervalArithmetic source code is hosted at https://github.com/FabriceSalvaire/python-interval-arithmetic

You have to solution to get the source code, the first one is to clone the repository, but if you are not familiar with Git then you can simply download an archive either from the IntervalArithmetic Pypi page (at the bottom) or the GitHub page (see clone or download button).

To clone the Git repository, run this command in a console:

git clone https://github.com/FabriceSalvaire/IntervalArithmetic.git

Then to build and install IntervalArithmetic run these commands:

python setup.py build
python setup.py install

3.7. Dependencies¶

IntervalArithmetic requires the following dependencies:

  • Python 3
  • PyYAML

To generate the documentation, you will need in addition:

  • Sphinx
