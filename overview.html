1. Overview¶

1.1. What is IntervalArithmetic ?¶

IntervalArithmetic is free and open source (*) interval arithmetic package for Python.

1.2. How is IntervalArithmetic licensed ?¶

IntervalArithmetic is licensed under the GPLv3.

1.3. Going further with IntervalArithmetic¶

The best way to know what you can do with IntervalArithmetic, and to learn it, is to look at the examples:

  • IntervalArithmetic Reference Manual

1.4. Which platforms are supported by IntervalArithmetic ?¶

IntervalArithmetic runs on Linux, Windows 64-bit and Mac OS X.

1.5. How to install IntervalArithmetic ?¶

The procedure to install IntervalArithmetic is described in the Installation Manual.

1.6. Which version of Python is required ?¶

IntervalArithmetic requires Python 3.
