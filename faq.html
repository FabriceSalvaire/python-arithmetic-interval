4. User FAQ¶

4.1. How to get help or report an issue ?¶

If you want to discuss or ask questions on IntervalArithmetic, you can subscribe and post messages on the IntervalArithmetic User mailing list.

There is actually three lists running on Google Groups (*):

User List
List for IntervalArithmetic users
Announce List
List for announcements regarding IntervalArithmetic releases and development
Devel List
List for developers of IntervalArithmetic

If you encounter an issue, please fill an issue on the Issue Tracker.

(*) Despite Google Groups has many drawbacks, I don’t have actually enough resources to run GNU Mailman or Discourse on my own IT infrastructure.
